# Homepage

A responsive portfolio homepage showcasing web development projects and skills. This project demonstrates modern web design principles with responsive layouts, custom typography, and engaging user interface.

## Table of Contents

- [Features](#features)
- [Technologies Used](#technologies-used)
- [Project Structure](#project-structure)
- [Getting Started](#getting-started)
- [Author](#author)

## Features

- **Responsive Design**: Mobile-first approach with breakpoints for tablets and laptops
- **About Section**: Professional introduction with links to GitHub and LinkedIn
- **Project Showcase**: Display of completed portfolio projects with screenshots
- **Custom Typography**: Integrated custom fonts (Caprasimo and Dancing Script) for unique branding
- **Modern CSS Grid**: Flexible, auto-fit grid layout for project cards
- **Accessible HTML**: Semantic HTML5 structure with proper alt text for images

## Technologies Used

- **HTML5** - Semantic markup and responsive images using `<picture>` element
- **CSS3** - Flexbox, Grid, custom fonts, and responsive media queries
- **Custom Fonts** - Caprasimo and Dancing Script for typography
- **SVG Icons** - GitHub and LinkedIn social links

## Project Structure

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The site is fully responsive and works on mobile, tablet, and desktop devices
